863. All Nodes Distance K in Binary Tree

用HashMap存住从根root到target 的path上所有节点离target的距离。

接着用dfs方法对每个节点进行扫描。如果map有当前的节点，说明在root-target的path上。直接将distance设置为该节点到target的距离。

如果当前节点不在map中且距离不是K（在target的子树里）,则下一次dfs将distance值加1。
