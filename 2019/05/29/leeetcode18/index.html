74. Search a 2D Matrix (Binary Search)

思路

二分查找的题，原题是2D矩阵，但其实不用那么复杂，考虑为一位数组就可以，利用mid/cols可以判断是第几小段，类似横向排布，所以matrix[mid/cols][mid%cols]方法将二维矩阵转换成了一维数组的问题。不错的idea。
