557. Reverse Words in a String III

思路

用空格来分割字符串，遇到space运行一次reverse，注意最后空格之后还有最后一部分字符串，所以要额外运行一次reverse。
reverse基本思想就是swap。

344. Reverse String

思路

时间复杂度O(n/2)，空间复杂度O(1),right place 改变顺序，不能用返回新String的方法了。二分法替换。

821. Shortest Distance to a Character

思路

two pass，左右各遍历一遍，找到min值。 将pos=-n纯粹是为了将其设的足够大。Brilliant。

867. Transpose Matrix

思路

矩阵转换题，注意新定义矩阵B的行列数和for的顺序，转置矩阵后MxN 变为NxM, 所以先j<N 后 i<M.

912. Sort an Array

思路

比较easy的题，但是要记住quick sort 的排序方法，先random选nums，分成<=>三部分，在各自递归调用，返回时注意加上m，中间值。

876. Middle of the Linked List

思路

非常重要的一种思路，fast和slow指针同时扫描，当fast到达结尾时，slow一定在middle(奇偶数定理可以证明).返回slow此时位置的后续指链。

872. Leaf-Similar Trees

思路

判断树最后各个节点是否相同，通常思路是用DFS遍历tree成list,然后比较两个list。但是比较麻烦。
这里将逐个叶节点进行比较，采用stack存储，其实这里遍历的方法类似pre-order，只不过不输出。
定义dfs函数，比较每个节点是否为最后一个节点，若左右不空，将其压入堆栈，若为空，返回最后节点的值，s1和s2的值进行比较。
