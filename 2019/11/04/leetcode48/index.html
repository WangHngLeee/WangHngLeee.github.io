209. Minimum Size Subarray Sum

1004. Max Consecutive Ones III

  1. sliding window方法。永远保证window里0的个数为K个，如果超过就要再用start开始遍历，保证0的个数始终为K，然后将最长的substring存为res。

1102. Path With Maximum Minimum Value

  1. PriorityQueue以大根堆的形式存住各个位置的值。
  2. 将每个位置以 {值，x坐标，y坐标} 的形式存入priority。以值为参考进行自定义compare。
  3. 在nx ny移动的部分，将新位置的值和之前cur的值进行比较，依次存入priorityqueue，最根部的则是这些最小值中最大的值。
  4. 以供下次循环时poll使用。poll出来的均为上一次扫描中最小值中的最大值。在更新位置前要判断是否为最后一个元素，如果是直接返回cur[0]。
