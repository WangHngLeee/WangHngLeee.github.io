Tree Problems

1104. Path In Zigzag Labelled Binary Tree

自下而上。在顺序颠倒的一行时，先计算出深度，再由深度计算出当前label距离该行最大值的距离remains，然后通过正常顺序计算出对应的parent。并且addFirst到res中。

366. Find Leaves of Binary Tree

recursive。 得到每一层距离叶节点的最大距离level，叶节点level为0，存在res[0]，自下而上recursive。将每个level的节点存入res。

897. Increasing Order Search Tree

tail记录最终结果里root的下一个节点。left分支中，root.left的下一个节点就是root父节点。并且将root.left 设为null。右分支在recursive时，在res赋值中，将当前root当作下一节点存入res。

1123. Lowest Common Ancestor of Deepest Leaves

只要当某个root的left 和 right 深度都为deepest时 ，说明该位置的subtree对称深度相同。那么直接返将当前的root赋值给lcn。

1110. Delete Nodes And Return Forest

dfs recursive。如果当前节点在delete名单里，看它左右节点是否为空，如果有不为null的将其子节点存入res。如果都为null则彻底删除该节点，返回null。

如果不含该节点，则返回该节点。
