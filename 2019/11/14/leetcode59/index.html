101. Symmetric Tree

951. Flip Equivalent Binary Trees

572. Subtree of Another Tree

545. Boundary of Binary Tree

左边界 - 左叶子 - 右叶子 - 右边界。

注意左右add root的时机不同。左边先add再递归。右边是先递归再add。

注意三种情况的函数写法。left /leaves /right

103. Binary Tree Zigzag Level Order Traversal

用zig来标记该行是否zig输出。如果是zig的一行，则将poll出来的 add(0,temp)到 temp的开头。否则的话就按顺序add到temp中。每个teamp存的是每一行的元素，最后将temp加入res中。

124. Binary Tree Maximum Path Sum

tricky的地方在于：max赋值那里，math里面的max是左右子树遍历后的max，而外面的max还是未更新的max。注意最后return左右中最大的加上root val，返回给上一层继续递归。

653. Two Sum IV - Input is a BST

two sum 变种。依旧用set存下出现过的node值。如果set中有node有等于k-root.val，则返回true。没有就继续左右递归。
