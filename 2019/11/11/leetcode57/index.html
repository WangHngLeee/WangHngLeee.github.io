863. All Nodes Distance K in Binary Tree

  1. find将在target之上的node存入map。在target subtree里的不赋值。
  2. dfs中，如果是target之上的节点，将dist更新为map中存的到target的距离，再进行dfs dist+1。
  3. 如果是target的subtree，则循环到target自身时，dist会变为0，然后dfs更新，当dist==k时，该节点也就是要存入res的节点。

987. Vertical Order Traversal of a Binary Tree

  1. 新建pair 类，用queue存放所有树节点，记录所有树节点x的最小值和最大值，之后遍历要用到。
  2. map 存放x坐标都相同的pai。然后用compare 从小到大sort。
  3. 然后通过x顺序遍历map，将相同x的值存进同一个list，再加入res中。
