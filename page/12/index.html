10. Regular Expression Matching

Hint

hard的一道题，也是判断.* 的题，
1, 如果p.charAt(j) == s.charAt(i) : dp[i][j] = dp[i-1][j-1];
2, 如果p.charAt(j) == ‘.’ : dp[i][j] = dp[i-1][j-1];
3, 如果p.charAt(j) == '’: (细分三种情况)
1 当p.charAt(j-1) != s.charAt(i) : dp[i][j] = dp[i][j-2] // a 表示为空
2 当p.charAt(j-1) == s.charAt(i) or p.charAt(j-1) == ‘.’:（细分三种）
dp[i][j] = dp[i-1][j] //a* 表示两个a
or dp[i][j] = dp[i][j-1] // a* 表示一个a
or dp[i][j] = dp[i][j-2] // a* 表示为空
