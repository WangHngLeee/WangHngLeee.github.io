863. All Nodes Distance K in Binary Tree

Question Link >>>

先定义find函数，用map存target上方节点距离target的距离，target下方不存。

再dfs遍历所有节点，将距离为k的存入res。如果map中有该点，则为target上方的点。直接将dist更新为map对应的value即可。如果没有则一步一步向下深入 dist+1.

222. Count Complete Tree Nodes

Question Link >>>

Complete Tree Wiki 定义

所以如果最左或者最右有一个空缺就不是complete tree。所以找最左最右的深度。

如果相同，则直接返回树 1<<深度 -1就是node的个数。<<相当于对深度取2的对数。

如果左右深度不相同则递归继续找各自左右子树并且加1（当前node）。
