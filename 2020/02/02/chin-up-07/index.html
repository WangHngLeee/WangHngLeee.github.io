97. Interleaving String

🔗Question Link

DP表表示当s1在第i个位置并且s2在第j个位置时，s3是否在第（i + j）个位置交织。第0位表示空字符串。

因此，如果s1和s2当前都为空，则s3也为空，并且被认为是交织的。如果只有s1为空，则如果先前的s2位置正在交织并且当前s2位置char等于s3当前位置char，则认为是交织。类似的想法也适用于s2为空的情况。当s1和s2都不为空时，如果我们从i-1，j到达i，j，则如果i-1，j已经交织并且i和当前s3位置相等，则为交织。如果我们从i，j-1到达i，j，则如果i，j-1已经交织并且j和当前s3位置相等。它是交错的。
