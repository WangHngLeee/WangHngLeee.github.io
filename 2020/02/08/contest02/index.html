1346. Check If N and Its Double Exist

🔗Question Link

First Solution

当时思路： 用set存每个num。然后遍历arr，找是否有当前num二倍数的存在。注意零的存在。

Optimized Solution

Tricky。直接在加入num之前先判断set里有无它的二倍数或者它的1/2数。如果没有则加入，有则直接返回true。
