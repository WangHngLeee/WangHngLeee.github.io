99. Recover Binary Search Tree

🔗Question Link

in-order深度遍历。然后依次pop，找到第一个pre大于cur的，则pre是第一个错误的地方，紧接着找第二个，如果之前first不为空，说明找到了，则把当前的给second。最后swap一下就可以了。

450. Delete Node in a BST

🔗Question Link

recursive.找到key后，判断左右是否为空，如果都不为空，则去右子树找最小值，找到后把当前root左树给到最小值的左树，返回最小值的右树。
Time : O(h) -> h is the height of tree.
