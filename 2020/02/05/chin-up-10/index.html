33. Search in Rotated Sorted Array

🔗Question Link

sort的数组被rotate后肯定有两部分递增的区域。mid>left说明mid在较大的递增一侧，否则是在较小的递增区。针对每个区，进行正常的binary search操作，对left和right操作。

由于用的是while left + 1 < right的写法，所以最后要先检查left是否等于target，然后再检查right。

79. Word Search

784. Letter Case Permutation

🔗Question Link

BFS Solution

把每个字母大小写情况全加进queue，然后依次往后循环遍历，最后把S中每个字母大小写的所有情况全部加入了queue。注意跳过为数字的char。最后queue中的个数应该是S字母长度的阶乘。

DFS Solution

1306. Jump Game III

🔗Question Link

Recursive Soluiton

jump是当前能跳的步数，然后将其变为-1。如果jumpto是0，则直接返回true。否则递归调用向左或向右跳的情况。

Iterative Solution
