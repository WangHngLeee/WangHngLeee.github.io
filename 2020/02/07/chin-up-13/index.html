200. Number of Islands

🔗Question Link

Using DFS . When we meet 1, we using dfs to find 1 in four directions. If it stop, it means we find an island and we can not move to the other 1. So we increase sum. THen we try the other position of 1 in the grid.

547. Friend Circles

🔗Question Link

类似island的题，由于是每个坐标代表一个人，这次只需要一行一行dfs就可以。seen检查是否访问过该人。对坐标值为1的地方一层层递归dfs即可。注意嵌套内dfs的 新person 值是当前 person 的 friend。
