437. Path Sum III

🔗Question Link

由于是BST，所以只需要通过大小判断就可以选择左右子树。如果当前root全部小于p,q则说明lca在右边的tree。若全小于p，q则在左边。如果大于p小于q则他自己本身就是，如果小于p大于q则是不可能事件，返回root。

257. Binary Tree Paths

🔗Question Link

不是BST，只是单纯的二叉树了。不能用大小边界来判断了。要分情况递归讨论。

  1. root为空或者 root只为pq中的一者，那么直接返回当前root。
  2. 递归找左右子树中pq的lca，如果left为空，说明只能在right，返回right。
  3. 反之返回left。
  4. 如果left和right1都不为null，说明p q 二者各自分布在左右中，所以返回当前root。
