今天学习了Go的基本语法和数据结构。感觉和Ruby有点像…

以下是和以往学习的语言有很大不同的地方。

声明类型

GO和java的类型声明正好相反，java类型写在变量前面，而Go写在变量后面。var name type = xxx.

Go语言的基本类型有：

bool

string

int、int8、int16、int32、int64

uint、uint8、uint16、uint32、uint64、uintptr

byte // uint8 的别名

rune // int32 的别名 代表一个 Unicode 码

float32、float64

complex64、complex128
